Drag & Drop Learning Game

A Unity-based interactive learning game focused on drag-and-drop matching with instant feedback. Designed for clarity, smooth UI flow, and replayability.

UnityC#UI/UXGame LogicFeedback

Step 1 — Drag & Drop

Players drag answers into slots. System checks if all slots are filled correctly.

Step 2 — Validation

Submit button validates each slot and gives immediate correct/wrong visual feedback.

Step 3 — Scoring & Timer

Points and countdown timer create challenge. Difficulty can change background & UI theme.
